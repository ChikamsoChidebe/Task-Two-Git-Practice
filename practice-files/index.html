Essential Git Commands

git init

Initialize a new Git repository

git add

Stage changes for commit

git commit

Save changes to repository

git push

Upload changes to remote

Git Workflow

  1. Create or clone repository
  2. Create feature branch
  3. Make changes
  4. Stage and commit
  5. Push to remote
  6. Create pull request
  7. Review and merge
